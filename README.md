# AI Games

Repo for games built with AI for the **Vibe Jam 2026** game jam.

## Vibe Jam 2026

A game jam where at least 90% of the code must be written by AI. Entries must be:
- **Web-only** — playable in the browser with no login/signup
- **Free-to-play** with its own URL
- **Instant load** — no loading screens or heavy downloads
- **Started during the jam** — no pre-existing games

ThreeJS is the recommended engine. Multiplayer is preferred but not required. Judged by game dev and AI community members.

Full rules are in [`rules.txt`](rules.txt).

## Games

### [Aura Flow](./aura-flow/) — *A Zen Kinetic Experience*

Control a liquid light orb through an infinite dark space. Navigate architectural rings, leaving a vivid glowing trail that paints the environment. Each ring shifts your color and adds a harmonic to the evolving soundscape. A meditative flow-state experience built with React Three Fiber, Zustand, and Web Audio API.

**Play:** [joeygibson.github.io/ai-games/aura-flow](https://joeygibson.github.io/ai-games/aura-flow/)  
**Details:** [`aura-flow/RUN.md`](aura-flow/RUN.md)
